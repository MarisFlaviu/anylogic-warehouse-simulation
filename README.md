# AnyLogic Warehouse Simulation

A warehouse operations simulation developed in AnyLogic as a university project.

## About the project

The model illustrates the main stages involved in handling goods inside a warehouse:

- unloading goods from incoming vehicles;
- transporting goods within the warehouse;
- directing goods toward the storage area;
- stacking and storing goods in designated locations;
- observing the movement of resources and materials throughout the process.

The simulation provides a visual representation of the warehouse flow and helps analyze how goods move from the unloading area to their final storage position.

## Technologies used

- AnyLogic
- Discrete-event simulation
- 3D modeling
- Process modeling
- Warehouse operations

## Project files

- `ModelBun.alp` – main AnyLogic simulation model;
- `3d/` – 3D resources used in the simulation;
- `warehouse-layout.png` – overview of the modeled warehouse layout.
